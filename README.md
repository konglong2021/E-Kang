## Installation

-Copy __.env.example__ file to __.env__ and edit database credentials there
-Composer install <br />
-npm install <br />
-npm run dev <br />
-php artisan key:generate <br />
-php artisan storage:link <br />
-php artisan migrate:fresh --seed <br />
-php artisan serve
