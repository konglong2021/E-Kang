## Installation

Duplicate .env.example and rename to .env <br />
-Do a configuration with laravel database  <br />
-Composer update <br />
-npm install <br />
-npm run dev <br />
-php artisan key:generate <br />
-php artisan migrate:fresh <br />
-php artisan serve
